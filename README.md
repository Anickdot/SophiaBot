# SophieBot

## How to run
- Create `.venv` virtual environment via 
  ```console
  python3 -m venv .venv
  ```
- Install required dependencies via
  ```console
  pip install -r requirements.txt
  ```

- Add `.env` file with environment variables:
  - CMC_ID - your [CoinMarketCap](coinmarketcap.com) API key
